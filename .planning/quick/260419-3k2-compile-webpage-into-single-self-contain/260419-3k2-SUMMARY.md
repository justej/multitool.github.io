# Quick 260419-3k2: Compile Webpage into Single Self-Contained File — Summary

**One-liner:** Node.js build script inlines CSS (base64 WOFF2 fonts) and concatenates 6 ES modules into a single 280 KB `dist/index.html` deployable to GitHub Pages.

## What Was Built

`build.js` — a zero-dependency Node.js script that:
1. Reads `style.css`, base64-encodes both WOFF2 fonts, and substitutes the `url('fonts/...')` references with `data:font/woff2;base64,...` URIs
2. Reads all 6 JS modules in dependency order, strips `import` and `export` keywords, and joins them into one script block
3. Replaces `<link rel="stylesheet">` and `<script type="module" src="app.js">` in `index.html` with the inlined content
4. Writes `dist/index.html` (280 179 bytes)
